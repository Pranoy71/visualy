## Installation

Requirements: PHP 8.0+, a web server (Apache/Nginx) and a browser.

-Local (PHP built-in)
-Download or clone the repo.
-From the project root, run:  php -S localhost:8000

-Open http://localhost:8000 in your browser.

## Quick Start

1. Paste any original text in Text A
2. Paste the modified text in Text B
3. Click "Compare Texts"
4. See highlighted differences instantly!
